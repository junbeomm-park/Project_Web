* 프로젝트 배경

  > **프로젝트 주제** : 관광지 조회 및 일정 관리 사이트.
  > **프로젝트 목적** : 포트폴리오 제작 및 협업을 통해 협동심 함양.
  > **프로젝트 개요** : JAva, JQuery, HTML5, CSS, Oracle, Spring 등을 기반으로 오픈 API를 활용한 웹사이트 제작.
  > **기대효과** : 실제 서비스와 연결해보며 프로그래밍 언어의 활용력을 높이고 필요한 새로운 지식들을 습득.

    1. 초기 기획과 달라진 기능 구현

    4. 관리자 페이지
